#### 문제설명

정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

<br>

#### 제한사항

-   n은 0 이상 3000이하인 정수입니다.

<br>

#### 입출력 예

| a   | return |
| --- | ------ |
| 12  | 28     |
| 5   | 6      |

<br>

#### **풀이**

```javascript
function solution(n) {
    var answer = 0
    for (let i = 1; i <= n; i++) {
        if (n % i == 0) {
            answer += i
        }
    }
    return answer
}
```

1. `for`문으로 `n`의 수 만큼 반복문을 돌린다.
2. 나머지가 0이면 답에 계속 더해준다.

<br>

#### **다른 사람 풀이 **

```javascript
function solution(n, a = 0, b = 0) {
    return n <= a / 2 ? b : solution(n, a + 1, (b += n % a ? 0 : a))
}
```
