# 📗 Node.js가 뭔데!
> Node.js®는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임


<br>

위 내용은 공식 문서에 정의되어 있는 내용으로 런타임은 프로그램 언어가 구동되는 환경을 말한다.   
즉 Node.js는 **Javascript를 실행 시킬 수 있는 환경을 만들어주는 툴**이다.
