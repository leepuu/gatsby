[Youtube] Youtube Player API + Swiper

2021-06-14

🎬 유튜브 슬라이드 제작하기

Youtube Player API와 Swiper를 사용하여 슬라이드를 제작해보도록 하겠습니다~

종류는 총 2가지로 스와이핑의 가능 유무에 따라 제작하도록 하겠습니다.

1️⃣ Swiping 불가능 슬라이드

1. 기본 슬라이드 - 슬라이드가 전환될 때 재생 X

<iframe> 태그 사용

HTML
<div class="box_area"> <div class="swiper-container"> <div class="swiper-wrapper"> <div class="swiper-slide" > <iframe src="https://www.youtube.com/embed/KqJ_AeSdBRM?enablejsapi=1" frameborder="0" allow="autoplay;" allowfullscreen></iframe> </div> <div class="swiper-slide" > <iframe src="https://www.youtube.com/embed/Gvq4bawFRiU?enablejsapi=1" frameborder="0" allow="autoplay;" allowfullscreen></iframe> </div> <div class="swiper-slide" > <iframe src="https://www.youtube.com/embed/v7Qybl26lZE?enablejsapi=1" frameborder="0" allow="autoplay;" allowfullscreen></iframe> </div> </div> <div class="swiper-pagination"></div> </div> <div class="swiper-button-prev"></div> <div class="swiper-button-next"></div> </div>
JavaScript
